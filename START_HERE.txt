方法 1：使用快速脚本（推荐）
----------------------------------
在当前目录运行：

Windows:
    .\push-to-github.bat

Linux/Mac:
    chmod +x push-to-github.sh
    ./push-to-github.sh

脚本会引导你：
1. 输入 GitHub 用户名
2. 配置远程仓库
3. 推送代码


方法 2：手动操作（标准流程）
----------------------------------
第 1 步：在 GitHub 创建仓库
    访问：https://github.com/new
    
    仓库名称：cursor-rules
    描述：🚀 Cursor AI 专用的 Java 开发规范集
    选择：Public（公开）
    注意：不要勾选任何初始化选项！

第 2 步：推送代码（替换 YOUR_USERNAME）
    git remote add origin https://github.com/YOUR_USERNAME/cursor-rules.git
    git branch -M main
    git push -u origin main

第 3 步：美化仓库（可选）
    - 添加 Topics 标签
    - 启用 GitHub Pages


方法 3：使用 GitHub Desktop
----------------------------------
1. 打开 GitHub Desktop
2. File -> Add Local Repository
3. 选择此目录：E:\code\idea\springboot-demo\cursor-rules-repo
4. Publish repository
5. 命名：cursor-rules，选择 Public
6. 点击 Publish

常见问题：
Q: 推送时要求密码？
A: GitHub 不再支持密码，需要使用 Personal Access Token

Q: 推送被拒绝？
A: 确保远程仓库是空的，没有 README 等初始化文件

Q: 如何生成 Token？
A: GitHub Settings -> Developer settings -> Personal access tokens
